功能特性

easykube可在5分钟内部署用于生产环境的Kubernetes集群，自动化安装、向导式配置、Master节点高可用、分布式存储等是easykube的基本能力。

裸金属架构

easyKube是基于Ubuntu 18.04和Kubernetes的开源容器云操作系统，采用自动化安装方式。支持纯物理服务器和私有云环境（VMware/Openstack/Cloudstack…）部署。

离线安装

easyKube集成了Docker、Kubernetes、DockerHub、CoreDNS、Flannel、dashboard等容器云环境所需的必要组件，开箱即用，无需联网下载。

易于使用

easyKube采用自动化安装和基于WEB的向导式配置界面，通过简单的点击鼠标即可完成系统初始化配置。

灵活部署

easyKube可部署为仅有一个Master节点的开发/测试集群，也可部署为具有三个高可用Master节点的生产集群。

支持生产环境

easyKube支持网卡绑定、虚拟IP、Master节点高可用等生产环境所需的必要特性。

分布式存储

easyKube集成GlusterFS分布式文件系统，可作为DockerHub的存储空间和Kubernetes的持久化存储。

横向扩展

easyKube最小化部署仅需1台物理或虚拟化云服务器，可根据业务需求随时向集群添加新节点。
